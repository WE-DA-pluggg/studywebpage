## 기능 요약 설명
* 회원가입/로그인을 통한 계정 정보 저장 및 관리
* 로그인 권한이 없더라도 메인 페이지 및 스터디 상세페이지 열람 가능
* 메인페이지에서 api로 간략한 설명과 사진을 요청하고 이것을 보내줌
* 상세페이지에서는 댓글 리스트를 보내주어 타 유저의 스터디 모집에 대한 반응 가능
